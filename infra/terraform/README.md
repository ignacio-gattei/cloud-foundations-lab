# Terraform/OpenTofu

Esta carpeta es una introduccion a Infrastructure as Code.

No provisiona recursos cloud reales. Sirve para practicar:

- estructura de archivos;
- variables;
- outputs;
- `fmt`;
- `validate`;
- conversacion tecnica sobre `plan`, `apply` y `state`.

## Comandos

```bash
terraform init
terraform fmt
terraform validate
```

O con OpenTofu:

```bash
tofu init
tofu fmt
tofu validate
```
